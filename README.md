Student Performance and Attendance Analyzer

An Exploratory Data Analysis (EDA) and visualization project focused on identifying student academic patterns, performance variance across courses, and the direct impact of classroom attendance on cumulative grades (CGPA).

Project Overview
Educational institutions handle large datasets containing academic and behavioral tracking metrics. This project implements an end-to-end data processing workflow using Python and Pandas to extract actionable institutional insights. By evaluating metric relationships, this system highlights performance distributions and tracks how absenteeism directly correlates with academic outcomes.

Tech Stack & Libraries

Language: Python

Environment: Google Colab / Jupyter Notebook

Data Manipulation: Pandas, NumPy

Data Visualization: Matplotlib, Seaborn

Key Implementations

Data Inspection & Preprocessing: Streamlining structural validation, handling data typing, and analyzing missing data variables using automated aggregation methods (df.info()).

Exploratory Data Analysis (EDA): Computing statistical summaries (df.describe()) and multi-variable frequency count distributions.

Data Visualization: Developing unified analytics dashboards featuring:

Histograms with Kernel Density Estimations (KDE) to observe CGPA density curves.

Box Plots to isolate academic variance across distinct courses.

Bar Charts to directly gauge average performance gaps based on attendance status.
